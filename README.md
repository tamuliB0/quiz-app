# quiz-app
Welcome to the Quiz App! This project is a simple, PHP-powered general knowledge quiz. 
It allows users to take a quiz, navigate through questions and view the results after completion.

# Features 
- Session management: Tracks user's progress through the quiz
- Navigation: Users can go back to previous question or move on to the next one 
- Results: After completing the quiz, users can view their answers and check if they were correct
